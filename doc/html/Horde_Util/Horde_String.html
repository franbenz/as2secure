method convertCharset [line 76]

	

    

    

		mixed convertCharset(
mixed
$input, string
$from, [string
$to = null])
    

    


	
		Converts a string from one charset to another.

Works only if either the iconv or the mbstring extension  are present and best if both are available.  The original string is returned if conversion failed or none  of the extensions were available.





Tags:





  

    return:  The converted input data.
  

  

    access:  public
  









	
        
Parameters:

    

    

          

        mixed  
        $input  
        The data to be converted. If $input is an an array,                       the array's values get converted recursively.
      

          

        string  
        $from  
        The string's current charset.
      

          

        string  
        $to  
        The charset to convert the string to. If not                       specified, the global variable                       $_HORDE_STRING_CHARSET will be used.
